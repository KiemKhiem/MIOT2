// Biên dịch ứng dụng
1. mkdir webcage && cd webcage && npm init --y
2. Trong file package.json nhập dòng lệnh npm install express moment socket.io
3. npm install -D nodemon
4. Tạo 1 file index.js chứa code phần Server
5. Tạo 1 folder tên public, folder này chứa các mục như:
	+ Các file HTML: chứa phần front-end với người dùng
	+ Các file CSS: chứa phần chỉnh sửa giao diện ( màu sắc, cỡ chữ,...)
	+ main.js: chứa code phần Client
6. npm run dev (chạy local)
7. Tạo liên kết với Firebase
8. Đưa ứng dụng lên mạng internet qua heruku
